Getting Started

Commands

  • npm run storybook - Run Storybook to view components interactively
  • ng serve - Start development server at localhost:4200
  • ng test - Run unit tests with Karma
  • ng build - Build for production

Clean Architecture Pattern

Each component follows a clean architecture structure:

  • Component File (*.component.ts) - Core component logic and UI
  • Stories File (*.stories.ts) - Storybook documentation and visual testing
  • Spec File (*.component.spec.ts) - Unit tests using Jasmine/Karma
  • Barrel Export (index.ts) - Centralized component exports
